monotransits

By Hugh Osborn

Code Repository

This is a public-facing page for all exoplanet code from Hugh Osborn (Personal page here).

This includes the Monotransit Catalogue, consisting of more than 100 single transiting planet candidates from CoRoT, Kepler, K2 and TESS.

It also includes resources for Cheops-TESS collaboration, namely info for each TOI which Cheops may observe.

Monotransit Plots

To help assess monotransit candidates, I have assembled all monotransits from Kepler, K2 and TESS in one list, and created interactive plots for every one.

Monotransit catalogue

A catalogue of monotransits across multiple space-based photometry missions, uniformly processed to determine which appear most planetary, and to provide their orbital parameters to the community.

CHEOPS/TESS TOI plots

To assist with the TESS-CHEOPS (CHESS) program, I have assembled all TOI data, calculated Cheops-specific information such as observability, and plotted each possible TOI here.
